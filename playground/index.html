[AVRP] Playground

Welcome to the Playground - the entertainment bot for Discord. The system provides users with different games, like the Roulette, Slot-machine, Wheel of Fortune and Dice games. For the best user experience bot also has a profile system, acheivements and badges system, in-game shop and a simple access to all information about Playground via a help command. Server's administrators are able to configure the bot: they can change the bot's language, roles allowed to use the bot or configure the bot, and channels the bot is allowed to be used in. Happy games!
